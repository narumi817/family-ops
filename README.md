# FamilyOps

家事・育児のタスクを記録し、ポイントで管理する家族向けログアプリケーション

## 📖 プロダクト概要

FamilyOpsは、家族で家事・育児のタスクを記録し、ポイントで管理するWebアプリケーションです。

- **家族単位でのタスク管理**: 家族ごとにタスクを登録・管理
- **ポイントシステム**: タスクごとにポイントを設定し、実行したタスクのポイントを累計
- **ログ記録**: 誰がいつどのタスクを実行したかを記録
- **家族招待機能**: メールで家族メンバーを招待して参加

## 🛠️ 技術スタック

### バックエンド
- **Rails 8** (APIモード)
- **PostgreSQL** (データベース)
- **RSpec** (テストフレームワーク)
- **FactoryBot** (テストデータ生成)
- **Resend** (メール送信)

### フロントエンド
- **Nuxt 3** (Vue.js フレームワーク)
- **TypeScript** (型安全性)
- **Tailwind CSS** (スタイリング)
- **Pinia** (状態管理)
- **Vee-Validate** (フォームバリデーション)

### インフラ・開発環境
- **Docker / Docker Compose** (開発環境)
- **Render** (本番環境)

## ✨ 基本機能

### 認証機能
- **ユーザー登録**: メールアドレス認証による新規登録
- **ログイン/ログアウト**: セッション管理による認証
- **家族招待**: メールで家族メンバーを招待

### タスク管理
- **タスク一覧**: 全家族共通タスク + 家族固有タスクの表示
- **タスク登録**: 家族固有のタスクを追加（カテゴリ: 育児/家事、ポイント: 0〜100pt）
- **ポイント設定**: タスクごとのポイントを家族単位でカスタマイズ

### ログ管理
- **ログ登録**: 実行したタスクを記録（実行日時、メモ付き）
- **ログ一覧**: 家族のログをタイムライン表示（実行者別の色分け）
- **ポイント表示**: 当日の累計ポイントを表示

### 家族管理
- **家族作成**: ユーザー登録時に自動で家族を作成
- **メンバー管理**: 家族メンバーの招待・参加管理

## 🔮 将来の機能（実装予定）

### 認証・招待機能の拡張
- **QRコードによる家族招待**: メール送信以外の招待方法として、QRコードで招待トークンを共有
- **OAuth認証（Google）**: Googleアカウントでのログイン・サインアップ
- **JWT認証**: トークンベースの認証方式（オプション）

### ログ管理の拡張
- **フィルタリング機能**: ログ一覧で実行者、タスク、日時で絞り込み
- **ログ編集・削除**: 登録済みログの更新・削除機能

### ポイントシステムの拡張
- **ゴール設定**: ポイントが何ポイント貯まったら達成できるゴールを設定
- **ポイント消費機能**: 累計ポイントと使用できるポイントを別管理し、ポイントを消費できる機能

### タスク管理の拡張
- **タスク削除機能**: 家族固有タスクの削除（論理削除 or 物理削除は要検討）
- **タスク編集機能**: 登録済みタスクの情報更新

### その他
- **ユーザー情報管理**: プロフィール編集、パスワード変更
- **統計情報表示**: より詳細な分析・可視化機能
